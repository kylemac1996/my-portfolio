Projects

Stock Search

The app is programmed to search any stock data by searching its ticker symbol. I used react to create this app because it offers the simplicity of HTML layout while using javascript all in one language. Class components were used along with react hooks to build the program. Once ticker is searched the app will pull up a card that shows important data about that company: name, symbol, percent change (red=loss, green=gain), year high and year low. To see project in action click download link (either fast or slow speed), click download now. Once downloaded enter npm install in your text editor to install dependencies, then run npm start to run application.

Robofriends

The program was created using react.js. This helps with the simplicity of putting all the files together. As opposed to the stock search, I used functional components to create robofriends. This app has a list of cards pulling from a set list of users. Each card has a picture of a robot, a name, and email address. There is a search bar at the top used to filter out the cards on the page. The webpage filters as you enter letters or erase them. If you would like to scroll the page to see the users on it, the search bar at the top is meant to scroll with the user. To see project in action click see live.
